按键，GPIO口A2复位为按键按下，添加防抖代码。按键按下即启动主程序（主程序见最后）。
运动模块，使用A6A7的TIM3计时器功能生成PWM信号，A4A5A6控制小车左侧两轮，A7B0B1控制小车右侧两个轮子。使用了H桥驱动模块，A4A5B0B1为正反转控制端口。
循迹模块：B5、B6、B7、B8、B9，为GPIO，设为接收，当收到数字信号为低电平时代表在黑色预定轨迹上。5个端口所控制的传感器模块在小车前方的从左向右1、2、5、4、3位置。尝试使用PID算法实现循迹功能。
超声波模块：USART3的B10为TX、B11为RX，通过输出一个大于10us的高电平，会收到一个带时间信息的高电平。计时器获得高电平的保持时间，通过公式“距离=时间✖342.63/2”算出距离，当距离小于10cm时代表**有障碍物**，报警。
人体感应模块：端口B12负责感知小车后方，B13负责小车前方。使用GPIO数字信号接收，当收到高电平且保持0.5s，则代表有**异常人员闯入**，报警。
位置信息：GPIO口A1接受数字信号，低电平时加一。创建一个数组，通过累加接收到的次数计算出小车的大致位置。
报警包含本地报警和联网报警，如下：
- 本地报警功能：B14置位点亮LED灯、B15置位蜂鸣器发声、A8**保持**置位播放录音。
- 联网报警功能：USART1的A9为TX、A10为RX，发送**报警原因**和位置信息。（本部分我还不太精通，可以先放放。）使用的ESP8266模块，A11复位时通信模块复位，A12连接通信模块的GPIO0悬空为工作模式、下拉为下载模式。
